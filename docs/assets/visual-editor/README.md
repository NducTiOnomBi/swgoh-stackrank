# Visual Editor Screenshot Standards

All Visual Editor documentation screenshots live inside this folder.

## Requirements

- **Format:** PNG only
- **Resolution:** Minimum width 1280px (scale higher if capturing ultra-wide monitors)
- **Placeholder Overlay:** When a real capture is not yet available, include legible text that reads `Placeholder â€“ replace with actual screenshot` centered on a solid background with 4.5:1 contrast ratio.
- **File Naming:** Use kebab-case summaries (e.g., `tier-grid-overview.png`, `character-details-fields.png`).
- **Alt Text:** Every reference in markdown must include descriptive alt text, not simply "screenshot".
- **Captions:** Place the caption immediately below the image using standard markdown text.

## Updating Placeholders

1. Capture the real screenshot following the requirements above.
2. Replace the placeholder PNG using the same filename to avoid documentation churn.
3. Update the caption text in the markdown file if the new screenshot highlights different callouts.
